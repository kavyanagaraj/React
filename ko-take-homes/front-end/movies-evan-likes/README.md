# Movies Evan Likes!

## Setup

1. Clone this repo.
2. `npm i` or `yarn`
3. `npm rum serve` or `yarn serve` to start the local API and file server
3. `npm run watch` or `yarn watch` to start the bundle watcher
4. Open [http://localhost:3000](http://localhost:3000)
